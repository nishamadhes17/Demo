Definition List

HTML
HTML stands for Hyper Text Markup Language.It is used to markup a content's meaning with tags.
CSS
CSS stands for Cascading Style Sheets.It is used to style the web page's look and feel.
JS
JS stands for Java Script.It is used to determine the behaviour of web page.
NodeJS
NodeJS is a server side programming language developed with javascript.Though we can't rule them all with one ring,we can code them all in one language.
